A beautifully designed, interactive, fan-made field manual collection for **No Man's Sky**. 

The **Traveller's Codex** provides four highly detailed, independent deep-dive manuals written for travellers at every stage of their journey — from first-time Overseers to veteran Admirals commanding billion-unit trade empires.

## 📚 The Manuals

The Codex is split into four primary modules, each featuring a unique atmospheric UI and custom styling to match its theme:

### 🏘️ [Manual I: Settlements](nms-settlements-guide.html) (Overseer's Field Guide)
Master the full settlement lifecycle. Learn to handle Sentinel threats, resolve settler disputes, direct construction projects, and build a settlement that generates passive income and rare rewards.
* **Key Topics:** Finding & Claiming, Building Cycles, Debt Management, Sentinel Defense, Population Happiness.
* **Theme:** Deep Void & Neon Cyan.

### 🚀 [Manual II: Fleet Operations](nms-freighters-frigates-guide.html) (Admiral's Command Manual)
Build and command a capital ship fleet. Acquire the perfect S-class freighter, kit it out with technology, and deploy frigates across the galaxy on automated expeditions. 
* **Key Topics:** S-Class Hunting, Freighter Upgrades, Fleet Base Building, Frigate Types & Traits, Expedition Strategy.
* **Theme:** Riveted Steel & Amber Glow.

### 📈 [Manual III: Commerce](nms-trading-crafting-guide.html) (Merchant's Market Manual)
Master the galactic economy. Know exactly what to sell, what to keep, and what to refine first.
* **Key Topics:** Economy System, Sell vs. Keep Ledger, Crafting Recipes (Stasis Devices), Nanite Farming (Runaway Mould), Passive Income Loops.
* **Theme:** Grid UI & Terminal Green.

### 🛸 [Manual IV: Vehicles & Craft](nms-ships-exocrafts-guide.html) (Vehicle Command Manual)
Every vehicle in No Man's Sky covered in detail. Learn to optimize your starships, multi-tools, and exocraft.
* **Key Topics:** All 9 Ship Classes (including Interceptors & Living Ships), Custom Ship Fabrication (Corvettes), Exocraft Overviews, Upgrade Adjacency Bonuses.
* **Theme:** Halftone Textures & Alert Red.

## ✨ Features & Technical Details

This project is built using pure **HTML, CSS, and Vanilla JavaScript**. It requires no build tools or frameworks, ensuring lightning-fast load times and easy local hosting.

* **Immersive Visuals:** Animated CSS backgrounds, SVG illustrations, breathing glows, and a dynamic HTML5 Canvas starfield on the index page.
* **Responsive Design:** Fluid layouts that adapt beautifully from ultra-wide desktop monitors down to mobile screens.
* **Zero Dependencies:** Fully self-contained codebase (with the exception of Google Fonts).
* **Typography:** Utilizes thematic fonts (*Orbitron, Share Tech Mono, Cinzel, Bebas Neue*) to match the rich sci-fi aesthetic of the game.

## 🚀 How to Use

1. Clone or download this repository.
2. Open `index.html` in your preferred modern web browser.
3. Browse the index hub and click on any of the guide cards to navigate to the specific field manuals.
